Baron Charge State

This application is made while interning at CERN. The application is used to calculate the charge state of the ions in the accelerator complex at CERN.

For instance if the user wants to accelerate Pb ions then this calculater is used to calculate the charge state distribution of the Pb ions after the ions has passed the carbon stripper foil. This stripper foil is located in the Linac3 accelerator short after the ions has been produced. The resulting distribution can then be used to determine what ions should be used in the coming accelerators. The application is made in Python and is hosted on Streamlit. P9: Study of Beam and Residual Gas Interactions at the CERN Accelerator Complex

Aalborg University - CERN internship | Aug 2023 - Jan 2024

The current Pb ion beam suffers from heavy losses in the CERN Ion Injector Chain, consisting of Linac3, LEIR, the PS, and the SPS. This study focuses on the interaction between the ion beam and residual gas molecules, specifically on electron loss and electron capture from these interactions. Estimating the cross-section for electron loss and electron capture together with the inverse relation between the cross-section and lifetime is the theoretical foundation for this work. By injecting additional residual gasses in the PS it was possible to validate a new cross-section estimator. The new cross-section estimator was used to conclude that beam-gas interactions are not the primary factor limiting the beam intensity in the tested rings (LEIR, the PS, and the SPS).

P8: Omission of Frequency Content in Efficient Energy Flow Models for Integrated Energy Systems

Aalborg University | Feb 2023 - May 2023

This project explores the optimization of integrated energy systems through the exclusion of frequency content, aiming to maintain an acceptable level of accuracy. Test results show that the majority of the frequency signal can be omitted when a high tolerance of error is acceptable, particularly for simulations involving predominantly low frequencies. In tests, it was found that the feasibility of the optimization problem was not compromised by omitting frequency content. To identify how much frequency content can be omitted the project suggests using energy difference since this can be found before the optimization. Future research includes reducing the dimensionality of frequency content and exploring the computational benefits attached to this, integrating energy storage into the optimization mode, and analyzing the complexity of increasing network size. By addressing these research areas, the project aims to improve the computational techniques required in an integrated energy system.

P7: Recurrent Neural Network for Estimating Mutual Information of a Gaussian Process

Aalborg University | Aug 2022 - Jan 2023

Estimating mutual information is a long-standing problem in statistics and machine learning. The current state-of-the-art method for estimating mutual information is the KSG. In this paper, we propose using a recurrent neural network to estimate mutual information. The data used to train and test the network is created by a Gaussian generative model, which allows us to calculate the mutual information between two random variables.

We evaluate the network on simulated data and find that it is an unbiased estimator of mutual information, for this specific data. Furthermore, we find that the residuals of the network have a statistically significantly lower variance than those of the KSG. From the test results, we conclude that the network outperforms the KSG on the generated data.

This paper suggests that using a recurrent neural network for the estimation of mutual information might be the next development in getting a better estimator than the current state-of-art KSG.

P6 (Bachelor thesis): Satellite Tracking Using Multi Hypothesis Tracking and Kalman Filter

Aalborg University | Feb 2022 - May 2022

The goal of this project is to track multiple satellites using simulated data provided by Weibel Scientific A/S. The abstract is included below:

In this project, the goal was to track satellites in low earth orbit using Hypothesis Oriented Multiple Hypothesis Tracking and the Kalman Filter. The data used for the project was simulated using a script provided by Weibel Scientific A/S. The tests were carried out with different amounts of spread on the satellite tracks and with different amounts of false alarms. The algorithms successfully tracked the satellites in most scenarios. P5: Estimation of Currents and Voltages in an Electrical Grid

Aalborg University | Aug 2021 - Jan 2022

In this project, the goal is to estimate currents and voltages in an electrical grid. This grid represents the structure of a small Danish town. Currents and voltages are being measured by smartmeters in households, but in this project, we use a general linear model to estimate these smartmeter measurements so that fewer measurement devices are needed. The abstract for this project can be read below.

In this project smartmeters are used to measure currents and voltages in an electrical grid, representing a small town, which is assumed to follow a tree structure. Having smartmeters at every position would be expensive, hence the objective is to estimate currents and voltages in an electrical grid, with as few smartmeters as possible. Using theory concerning circuit analysis, the complex Gaussian distribution, optimization, complex differentiation, and statistics, a general linear model, which satisfies circuits laws, was derived. Assuming that we initially have smartmeters at every node in the tree structure we were, using the statistical model, able to remove approximately 30% of these smartmeters. Furthermore, we found that after replacing the measurements with estimates our model were not very tolerant towards defect smartmeters. It was, however, robust when detecting an increased impedance.

P4: Siren Detection

Aalborg University | Feb 2021 - May 2021

The purpose of this project was to design an algorithm to detect sirens in traffic. One of the learning goals for this project was to study the two types of digital filters; finite impulse response filters, and infinite impulse response filters. Below is the abstract for this project:

Sirens from emergency vehicles play an important role in traffic since they warn people of hazardous situations. Over time it has become more difficult to hear sirens, due to the improvement of noise reduction in car cabins and distractions such as music. This project focuses on the design of a detection algorithm that can be included as a vital part of a complete siren detection system. For the frequency analysis of different sirens, a short-time Fourier transform is used. Sirens used in this project change continuously in frequency and are periodic in frequency over time. Furthermore, when constructing a type I band-pass FIR window filter, it is used that the frequency range of most sirens is between 500Hz and 1500Hz. Based on this information a database is built. For the algorithm methods of estimating the period and minimum and maximum frequencies for sirens are developed. The robustness of the algorithm is tested by adding synthetic noise and interfering noises that appear in traffic. The tests showed that the detection algorithm worked as desired, but somewhat inconsistent. For further research, the database can be made with more characteristics of the sirens than the three used in this project in order to make the algorithm more consistent.

P3: Inverse Pendulum

Aalborg University | Aug 2020 - Jan 2021

The goal of this project is to stabilise an inverted pendulum. This is done using Lagrange mechanics, stability analysis, and control theory.

Abstract:
The goal of this project has been to stabilise the inverted pendulum on a cart. This is done by describing the physical system with a linearised model. To create this model, equations of motion were derived using Lagrangian mechanics. The stability of the system is described using Lyabunov’s indirect method, and then controllability and observability were used to introduce state feedback control, in order to stabilise the inverted pendulum. When implementing the control theory it was found that controlling the system could be done by choosing certain eigenvalues for the linear model. From the simulation it was seen that some choices of eigenvalues could stabilise the pendulum quickly, but with the consequence of a high peak in force. Other sets resulted in a longer time to stabilise the pendulum, less force, and a longer movement along the rail for the cart, to do so.

P2: Fingerprint Verification

Aalborg University | Aug 2020 - Jan 2021

This project used Linear Algebra, Wavelets, Poincaré Index, and the Pearson Correlation Coefficient to verify fingerprints. All algorithms were implemented in Python.

Abstract:
This project concerns fingerprint verification. A DB4 wavelet transform is used to remove Gaussian white noise from fingerprint images. Moreover, the Poincaré index is used to classify the fingerprints. The verification part is done with the Pearson correlation coefficient (PCC). Altogether, the wavelet transform did a good job removing noise, and the Poincaré index is a good classifier for images with a minimum amount of noise. The results of the implemented code was that it were able to recognize fingerprints to some extent. This was done by first classifying the fingerprint, and then comparing the PCC. The fingerprint were considered a match, if the PCC were above a set acceptance level.
