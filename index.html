P6 (Bachelor thesis): Satellite Tracking Using Multi Hypothesis Tracking and Kalman Filter

Aalborg University | Feb 2022 - May 2022

The goal of this project is to track multiple satellites using simulated data provided by Weibel Scientific A/S. The abstract is included below:

In this project, the goal was to track satellites in low earth orbit using Hypothesis Oriented Multiple Hypothesis Tracking and the Kalman Filter. The data used for the project was simulated using a script provided by Weibel Scientific A/S. The tests were carried out with different amounts of spread on the satellite tracks and with different amounts of false alarms. The algorithms successfully tracked the satellites in most scenarios. However, at a low signal to noise ration, the lack of detections caused issues for both algorithms.

P5: Estimation of Currents and Voltages in an Electrical Grid

Aalborg University | Aug 2021 - Jan 2022

In this project, the goal is to estimate currents and voltages in an electrical grid. This grid represents the structure of a small Danish town. Currents and voltages are being measured by smartmeters in households, but in this project, we use a general linear model to estimate these smartmeter measurements so that fewer measurement devices are needed. The abstract for this project can be read below.

In this project smartmeters are used to measure currents and voltages in an electrical grid, representing a small town, which is assumed to follow a tree structure. Having smartmeters at every position would be expensive, hence the objective is to estimate currents and voltages in an electrical grid, with as few smartmeters as possible. Using theory concerning circuit analysis, the complex Gaussian distribution, optimization, complex differentiation, and statistics, a general linear model, which satisfies circuits laws, was derived. Assuming that we initially have smartmeters at every node in the tree structure we were, using the statistical model, able to remove approximately 30% of these smartmeters. Furthermore, we found that after replacing the measurements with estimates our model were not very tolerant towards defect smartmeters. It was, however, robust when detecting an increased impedance.

P4: Siren Detection

Aalborg University | Feb 2021 - May 2021

The purpose of this project was to design an algorithm to detect sirens in traffic. One of the learning goals for this project was to study the two types of digital filters; finite impulse response filters, and infinite impulse response filters. Below is the abstract for this project:

Sirens from emergency vehicles play an important role in traffic since they warn people of hazardous situations. Over time it has become more difficult to hear sirens, due to the improvement of noise reduction in car cabins and distractions such as music. This project focuses on the design of a detection algorithm that can be included as a vital part of a complete siren detection system. For the frequency analysis of different sirens, a short-time Fourier transform is used. Sirens used in this project change continuously in frequency and are periodic in frequency over time. Furthermore, when constructing a type I band-pass FIR window filter, it is used that the frequency range of most sirens is between 500Hz and 1500Hz. Based on this information a database is built. For the algorithm methods of estimating the period and minimum and maximum frequencies for sirens are developed. The robustness of the algorithm is tested by adding synthetic noise and interfering noises that appear in traffic. The tests showed that the detection algorithm worked as desired, but somewhat inconsistent. For further research, the database can be made with more characteristics of the sirens than the three used in this project in order to make the algorithm more consistent.

P3: Inverse Pendulum

Aalborg University | Aug 2020 - Jan 2021

The goal of this project is to stabilise an inverted pendulum. This is done using Lagrange mechanics, stability analysis, and control theory.

Abstract:
The goal of this project has been to stabilise the inverted pendulum on a cart. This is done by describing the physical system with a linearised model. To create this model, equations of motion were derived using Lagrangian mechanics. The stability of the system is described using Lyabunov’s indirect method, and then controllability and observability were used to introduce state feedback control, in order to stabilise the inverted pendulum. When implementing the control theory it was found that controlling the system could be done by choosing certain eigenvalues for the linear model. From the simulation it was seen that some choices of eigenvalues could stabilise the pendulum quickly, but with the consequence of a high peak in force. Other sets resulted in a longer time to stabilise the pendulum, less force, and a longer movement along the rail for the cart, to do so.
